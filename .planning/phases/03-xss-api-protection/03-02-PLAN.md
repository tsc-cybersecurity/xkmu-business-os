STEP 1 — Create drizzle/migrations/0031_api_key_scoping.sql:

```sql
-- Migrate api_keys permissions from legacy format to module:action format.
-- All existing rows receive ['*'] for backward compatibility.
-- New keys created via Admin UI will receive explicit scope arrays.
UPDATE api_keys
SET permissions = '["*"]'::jsonb
WHERE permissions IS NULL
   OR (permissions != '["*"]'::jsonb AND permissions::text NOT LIKE '%:%');
```

Note: The WHERE clause leaves alone any rows that already have ['*'] or already use module:action format (contains ':'). The two UPDATE pattern from research is simplified here — a single UPDATE with the combined condition achieves the same result more cleanly.
