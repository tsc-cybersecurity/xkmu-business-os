```typescript

}
```

Implementierung:
1. DB-Query mit groupBy(sopDocuments.maturityLevel)
2. countMap aus dem Ergebnis bauen (null-Level ignorieren)
3. MATURITY_LABELS = { 1: 'Anfaenger', 2: 'Grundkenntnisse', 3: 'Kompetent', 4: 'Fortgeschritten', 5: 'Experte' }
4. Array fuer Level 1-5 erzeugen mit countMap-Lookup (Default 0)
5. totalSops = Summe aller Counts
6. return apiSuccess({ maturityDistribution, totalSops })

Imports: `db` from '@/lib/db', `sopDocuments` from '@/lib/db/schema', `eq, and, isNull, sql` from 'drizzle-orm', apiSuccess/apiServerError, withPermission, NextRequest.
